We will provide you with a build in rar or zip files, you can install the new program by extracting files.

Once you got the build go to your browser ( we support Chrome, Edge, Opera, etc...)
Right now, I will show you how to install it on Chrome - you can do the same with other browsers.

Instructions:

  1. Go to Chrome tab
  2. On the top-right, click on three dots (vertical ellipsis)
  3. Find More tools then Extensions

  1. Make sure developer mode is ON

  1. Click on Load unpacked button

  1. Once you click, it will open a file explorer - navigate to where you have stored/extracted Build folder.

  1. Now you will see PM Designer and you are ready to get started!

  1. After you have installed PM Designer, you will see extention button in the toolbar and and you'll be able to see/pin PM Assist there.

  1. Now you're one click away to open PM Designer...
